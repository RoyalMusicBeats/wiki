# NIC Performance Fix

## Disable TSO, GSO, and GRO

```bash
ethtool -K enp0s31f6 tso off gso off gro off
ethtool --set-eee enp0s31f6 eee off
```

## Update GRUB Configuration

1. Open the GRUB default file:
```bash
nano /etc/default/grub
```

2. Modify the `GRUB_CMDLINE_LINUX_DEFAULT` line:
```
GRUB_CMDLINE_LINUX_DEFAULT="quiet pcie_aspm=off intel_idle.max_cstate=1"
```

3. Update GRUB and reboot:
```bash
update-grub
reboot
```
